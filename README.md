# Screenshots 自动整理工具

自动整理 Windows OneDrive Screenshots 文件夹中的截图文件。

## 功能特性

- ✅ **自动定时整理**：每个整点的01分自动执行（如 16:01, 17:01）
- ✅ **智能归档**：将3天前同一时段的截图自动归档到对应文件夹
- ✅ **系统托盘常驻**：程序在系统托盘运行，不占用桌面空间
- ✅ **图标状态提示**：
  - 有新文件夹时显示"有.png"图标（绿色）
  - 无新文件夹时显示"无.png"图标（灰色）
- ✅ **手动执行**：右键菜单提供"立即马上手动执行一次"选项，方便测试

## 工作原理

程序会检测 `~\OneDrive\图片\Screenshots` 目录下的图片文件。

**时间计算逻辑**：
- 假设当前时间是 `2025-12-04 17:03`
- 程序会检查是否有在 `2025-12-01 16:00-17:00` 这个时间段创建的图片
- 如果找到符合条件的图片，会创建名为 `2025-12-01 16:00-17:00` 的文件夹
- 将这些图片移动到该文件夹中

## 安装依赖

```bash
pip install PyQt5
```

## 运行程序

```bash
python screenshot_organizer.py
```

## 右键菜单功能

- **立即马上手动执行一次**：手动触发一次检查和整理
- **打开Screenshots文件夹**：快速打开截图目录
- **关于**：显示程序信息
- **退出**：关闭程序

## 文件说明

- `screenshot_organizer.py` - 主程序文件
- `有.png` - 有新文件夹时的托盘图标
- `无.png` - 无新文件夹时的托盘图标

## 注意事项

1. 确保 `~\OneDrive\图片\Screenshots` 目录存在
2. 程序只处理图片文件（.png, .jpg, .jpeg, .gif, .bmp, .webp）
3. 程序根据文件的创建时间（Windows 的 st_ctime）来判断
4. 程序会在控制台输出详细的运行日志，方便调试
